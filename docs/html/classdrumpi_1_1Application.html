Detailed Description

Main application.

This is the main application for the DrumPi program. Its setup and run methods are called when the application is started.

◆ setup()

void Application::setup ( )

Sets up the application.

This method is called on startup to perform various set up tasks, including connecting the Application to the KeyboardInput as a callback, resetting the sequencer and display, and loading the drum sample bank.
