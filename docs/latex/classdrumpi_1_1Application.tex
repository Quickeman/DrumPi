\doxysubsection{Detailed Description}
Main application. 

This is the main application for the Drum\+Pi program. Its \mbox{\hyperlink{classdrumpi_1_1Application_a31dd3f7f8fba9fc2fd32ceaff6ce36e7}{setup}} and \mbox{\hyperlink{classdrumpi_1_1Application_a68965449404743bf1add056784d6cf81}{run}} methods are called when the application is started.
