\doxysubsection{Detailed Description}
\mbox{\hyperlink{classdrumpi_1_1clock_1_1Metronome}{Metronome}} class, similar to \mbox{\hyperlink{classdrumpi_1_1clock_1_1Clock}{Clock}} but can operate in B\+PM. To use, create a class that inherits from this and override the \mbox{\hyperlink{classdrumpi_1_1clock_1_1Clock_ade9259c06e6b90bbd92e155a2506d3a1}{tick}} method to set the functionality.
